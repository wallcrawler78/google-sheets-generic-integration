Configure BOM Position Attribute

Configure which BOM attribute should be used to track rack positions when pushing to Arena. This works like reference designators for electrical components - each rack will be tagged with its positions (e.g., "Pos 1, Pos 3, Pos 8").

Loading attributes from Arena...

How it works:
When you push a POD structure to Arena, each Rack on a Row's BOM will automatically be tagged with its position names from the Overview sheet.
Select BOM Attribute for Position Tracking
Select a text-type BOM attribute from Arena to store position information. If no suitable attribute exists, you'll need to create one in Arena first.
Position Name Format
Use {n} as a placeholder for the position number (e.g., "Pos {n}" becomes "Pos 1, Pos 2, Pos 3")
Example Output:
Pos 1, Pos 3, Pos 8
